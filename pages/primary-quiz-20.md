## 1.
<!--- 2012 (4) --->
::: problem id=1_1 marks=1
A blackboard has a total unfolded length of 6m.  The middle section is 3m long.  The two side sections are equal in length.  
 
![](/resources/primary-quiz-20/1-blackboards.png){image align="center"} 

How long is the section labelled with a question mark? 

* [ ] 1
* [ ] 1.25m
* [x] 1.5m
* [ ] 1.75m
* [ ] 2m
{.col-5}

---

The parts that fold into the middle must be equal in length and add up to the same length as the middle section.  

Therefore, these sections are both 3 ÷ 2 = 1.5 m.
:::


## 2.
<!--- 2014 (9) --->
::: problem id=2_1 marks=1
The little witch is taking part in a broomstick flying competition that is carried out in 5 rounds.  

The times in which she crossed the starting line can be seen in the table.  

![](/resources/primary-quiz-20/2-tables.png){image align="center"} 

Which round was her fastest? 

* [ ] 1st
* [x] 2nd
* [ ] 3rd
* [ ] 4th
* [ ] 5th
{.col-5}

---

We find the difference between the times after each round.  

The smallest difference is 10:54 – 10:26 = 28 min which is after Round 2.  
:::


## 3.
<!--- 2012 (14) --->
::: problem id=3_1 marks=2
The positive whole numbers are to be painted.  

1 is red, 2 is blue, 3 is green, 4 is red, 5 is blue, 6 is green and so on.  

Which colour(s), can the sum of a red and a blue number have?

* [x] Green only
* [ ] Red only
* [ ] Blue only
* [ ] Red or blue
* [ ] Impossible to say
{.col-5}

---

Red numbers are always one more than a multiple of 3.  

Blue numbers are always two more than a multiple of 3.  

Adding these together will always give a multiple of 3, which will therefore be green.  
:::


## 4.
<!--- 2014 (16) --->
::: problem id=4_1 marks=2
The points A, B, C, D, E, F are situated on a straight line in this order.  

The following distances on this line are known: *AF* = 35, *AC* = 12, *BD* = 11, *CE* = 12 and *DF* = 16.  

How long is the distance *BE*? 

* [ ] 13
* [ ] 14
* [ ] 15
* [x] 16
* [ ] 17
{.col-5}

---

It is difficult to solve this problem without drawing a diagram.  

![](/resources/primary-quiz-20/4-lines.png){image align="center"} 

Since AF = 35 and AC = 12, we know that CF = 35 – 12 = 23.  

Then, CE = 12 so that EF = CF – CE = 23 – 12 = 11.  

Next, DE = DF – EF = 16 – 11 = 5 and hence CD = CE – DE = 12 – 5 = 7.  

Finally, BC = BD – CD = 11 – 7 = 4.  

Putting all this together, we have that BE = BC + CD + DE = 4 + 7 + 5 = 16.  
:::


## 5.
<!--- 2016 (24) --->
::: problem id=5_1 marks=3
Two three-digit numbers are made up of six different digits.  

The first digit of the second number is twice as big as the last digit of the first number.  

Note: 0 is also a digit but cannot be the first digit of a number!  

How big is the smallest possible sum of the two numbers? 

* [ ] 301
* [ ] 535
* [x] 537
* [ ] 546
* [ ] 552
{.col-5}

---

We could have that the last digit in the first number is 1 and the first digit of the second number is 2, or we could have 2 and 4. Other possibilities, such as 3 and 6, will make for a larger sum.  

We fill in the other digits with the smallest numbers available (note that we could swap the middle digits in the first and second numbers).  

In the first case, we have 301 + 245 = 546.  

In the second, we have 102 + 435 = 537, which is the smaller sum.  
:::
